How to run:
	Make sure staff.txt file is located in src folder
	Make the directory of Terminal the src folder
	Compile EcsSim in Terminal by entering the command "javac EcsSim.java"
	Run the simulation by entering the command "java EcsSim staff.txt 1000 10" where 1000 is the integer budget, 10 is the integer number of years you want to run the simulation and staff.txt is the staff text file configuration
